# 📢 Automated LinkedIn Daily Publisher (AI News Digest)

Reads unprocessed AI news from Google Sheets, generates a 
newsletter-style summary using OpenAI, and auto-publishes 
it as a LinkedIn post — fully automated, daily.

## 🔧 Tools Used
- Make (formerly Integromat)
- Google Sheets
- OpenAI (ChatGPT)
- LinkedIn API
- Make Iterator + Array Aggregator

## ⚙️ How It Works
1. **Trigger**: Scheduled daily, reads Google Sheets for unprocessed AI news rows
2. **Array Aggregator + Iterator**: Batches all pending news
3. **OpenAI**: Generates a concise LinkedIn post summarizing the day's AI news
4. **LinkedIn**: Publishes the post automatically
5. **Google Sheets**: Marks rows as PROCESSED_AI to avoid duplicates

## 📈 Business Impact
- Consistent daily LinkedIn presence with zero manual effort
- Increased profile visibility and reach
- Demonstrates AI + automation expertise to potential clients
